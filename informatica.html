Java

Warlock

Questo è Warlock (precedentemente RSAMessages), un piccolo programma Java che offre un'elementare forma di crittografia per file di testo. Per ora è pensato per essere eseguito attraverso un'IDE, più nello specifico IntelliJ, ma in futuro potrò fornire un'implementazione più user-friendly, magari direttamente sotto forma di applet qui sopra (posto che github pages lo permetta, cosa di cui non sono sicuro). Potete usarlo per scambiarvi messaggi con i vostri amici, e perchè no, farmi sapere come vi trovate ed eventuali critiche o suggerimenti.
